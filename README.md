# Cursor-Inspired Developer Tool Landing Page (HTML & CSS)

This project is a desktop-first static recreation of a developer tool landing page inspired by the official Cursor website.
The objective was visual and structural accuracy, not creativity or interactivity.
The implementation strictly follows the assignment constraints and focuses on layout, typography, spacing, and overall visual hierarchy.

# Assignment Objective

Recreate a developer-focused landing page inspired by cursor.com, emphasizing:
Page structure and section layout
Typography and font usage
Color palette and spacing
Overall visual hierarchy

# No JavaScript, frameworks, animations, or responsive behavior were used.
